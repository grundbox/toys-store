How to start
===
1) clone repository
        
        git clone https://github.com/pathliving/toys-store.git toys-store

2) go to 'toys-store' directory

        cd toys-store

3) install dependencies

        npm i

4) start project

        npm start

5) open this link in your browser

        http://localhost:8080


    or click on this [link](http://localhost:8080)
